申请者隐私声明

适用于工作应聘者或合约服务

最后更新日期：2022 年 3 月 7 日

适用范围

本隐私声明适用于个人资料的处理，处理 Block（简称“我们”或“我们的”）收集的关于工作应聘者或合约服务（均称为“您”）的个人资料。对于欧盟和澳大利亚居民，本文还说明了您的资料保护权利，包括反对 Block 执行某些处理的权利。“您的选择和权利”部分阐述了关于您的权利以及如何行使权利的更多信息。（对于中国居民，本文还说明了我们如何收集、处理和保护您的个人资料、您的权利以及如何行使权利。）

当我们收集个人资料时，若我们认为提供相关和适时的信息会有所帮助，我们也可能会为您提供其他信息。

我们收集关于您的哪些信息

  • 个人身份和通讯信息：您的姓名、家庭住址、电话号码；电子邮件地址；出生日期、性别、国民保险号码和移民身份；
  • 申请相关信息：简历/履历 (CV)、求职信、工作经历、教育经历、资历和技能、推荐人联系信息、职位偏好、调职意愿、期望薪资、兴趣和抱负以及背景筛选信息（如果相关）；以及
  • 敏感信息：例如，若我们需要做出任何合理的调整，即可获取您的健康和残障相关信息。

以上信息从各种来源收集而来，但主要直接来自于您。您通常会在申请工作时提供此信息。我们还可能从第三方获得一些信息，例如，前任雇主的推荐信、面试和评估的结果、税务机关，或我们雇用第三方进行背景调查的情况（在适用法律允许的情况下）。此外，关于您的更多信息可能来自公共资源，比如 LinkedIn。

我们为何收集、使用和存储此信息

我们出于下文阐述的原因收集、使用和存储您的个人资料。

若 Block 有必要与您签订雇佣合约。

  • 就招聘做出明智的决定并评估您是否适合该职位。
  • 补偿您在申请过程中产生的任何约定费用。

必要时遵守法律义务。

  • 评估您是否适合担任受监管职位相关事宜（如适用）。

必要时为了以下所列 Block 的合法利益，以及我们的利益优先于您的资料保护权利时

  • 改善我们的招聘过程和活动
  • 核实您提供的详细信息并进行就业前背景调查；
  • 与您联系，了解 Block 未来的职业机会。
  • 保护我们的合法商业利益和合法权利，包括与法律索赔、合规、监管、审计和调查（包括与法律程序或诉讼相关的此类信息的披露）以及其他道德和合规报告要求有关的使用。
  • 根据适用法律分析和监测劳动力的多样性。例如，这包括遵守平等就业机会法。

若我们需要信息才可与您签订雇佣合约，或根据适用的就业、社会保障、社会保护或适用于我们的其他法律要求我们提供信息时，则必须提供信息。若您不向我们提供您的个人资料，我们将无法与您签订合约，或者我们可能需要撤销我们对某些职位（比如受监管职位）的聘用意向书。您可以任选提供所有其他信息。若您在上述情况中未提供相关资料，我们会在当时向您说明后果。

关于个人资料特定处理的其他信息可能会就地通知您，或者按照适用政策的规定通知您。

下文阐述了我们可能处理的敏感或特殊类别的个人资料。若适用的资料保护法律要求我们基于特定的合法理由处理此类特殊类别的个人资料，我们会根据以下任一依据进行相应处理：

  • 在适用法律允许的范围内，出于履行您或 Block 在就业法、社会保障和社会保护法领域的义务以及行使权利的目的，有必要进行处理；

  • 在适用法律允许的范围内，为了评估您的工作能力、医疗诊断或提供医疗保健或治疗，有必要进行处理；

  • 由于公共卫生领域的公共利益的原因，比如防范对健康的严重跨境威胁或确保医疗保健及医疗产品或医疗设备的高质量和安全标准，有必要进行处理；

  • 若您在生理上或法律上不具备表示同意的能力（例如，在公共卫生危机或医疗突发事件等特殊情况下），为了保护您或他人的切身利益，有必要进行处理；或

  • 对于建立、行使或辩护法律索赔，有必要进行处理。

核实和背景调查

对于某些职位，我们有必要核实您提供的详细信息（例如，您的身份、工作经历、学历和专业证书相关信息），并在法律允许的情况下进行就业前背景调查（例如，犯罪前科、财务状况、全球观察名单和性犯罪者记录相关调查）。调查级别将取决于您的职位，特别是您是否将担任受监管职务，并且调查将在招聘过程中尽可能后期进行，通常仅在您被选中担任该职位后进行。如果您的申请成功，我们将提供关于相关调查的更多信息，并将在完成此类调查之前获得任何必要的同意。

我们如何共享您的个人资料

我们将在您申请的职位或管理您的工作或任务所需的范围内，与 Block 的其他员工或订约人共享您的个人资料。

在招聘过程中，您的个人资料将与根据合约向 Block 提供服务的公司共享，比如背景调查。

如果出于上述目的需要、法律强制要求、或需要根据适用法律保护我们的合法利益，您的个人资料也将与政府机关和/或执法官员共享。

如果 Block 业务被出售或与其他业务整合，或者 Block 获得第三方融资，您的详细信息可能会披露给我们的顾问和任何潜在买方、贷方和/或其顾问。此外，在 Block 企业为买方的收购中，您的详细信息可能会与卖方和/或其顾问共享。

我们保留您的个人资料的期限

如果您被录用或聘用：根据适用法律，为了 Block 的合法利益，我们将仅在需要的期限内保留您的个人资料，用于招聘过程的目的，并且在此过程完成后，继续保留一段适当的时间，以便能够处理与申请过程相关的任何法律索赔。在此期限之后，我们将采取措施删除您的个人资料，或者以无法再识别您身份的形式保存资料。如果您成为 Block 员工，您提供的个人信息将成为员工档案的一部分，以后可能用于管理雇佣关系。

如果您没有被录用或聘用：我们将保留您的个人资料最长 2 年时间，以便我们在未来的招聘或聘用过程中考虑您。如果您不希望我们存档您的详细信息，请告知我们。

您的选择和权利

您有权要求 Block 提供您的个人资料的副本；更正、删除或限制处理您的个人资料；以结构化、机器可读的格式访问/获取您提供的个人资料，以及要求我们在技术上可行的情况下将其传输给其他管理机构。此外，您可以在某些情况下（特别是在我们不必处理资料以满足合约或其他法律要求的情况下）反对处理您的个人资料。如果您反对，我们必须停止处理，除非我们可以证明有令人信服的合法理由进行处理，该理由须高于您的利益、权利和自由，或者我们需要处理资料以建立、行使或辩护法律索赔的情况。在我们依赖合法利益作为处理依据的情况下，我们认为我们可以证明此类令人信服的合法理由，但我们会单独考虑每种情形。若我们已征求您的同意，您可以随时撤销同意。如果您要求撤销同意 Block 处理您的资料，这不会影响当时已经进行的任何处理。

这些权利可能会受限，例如，如果满足您的要求会泄露关于他人的个人资料，或者如果您要求我们删除法律要求我们保留或有令人信服的合法利益保留的信息。如果您有未解决的疑虑，您有权向您居住或工作所在国家/地区的资料保护机构投诉。

如需关于您的权利的更多信息或要行使您的任何权利，请联系 privacy-eu@Blockup.com 或 privacy-au@Blockup.com

跨国传输

英国和欧盟居民

出于上述目的，Block 成员和我们的服务提供商（例如筛查提供商）将在欧洲经济区（简称“EEA”）以外的管辖区处理或访问您的个人资料，这些服务提供商所在的管辖区没有与 EEA 等效的资料保护法律。

当我们将您的资料传输到 EEA 以外的 Block 附属机构时，我们使用包含欧盟委员会批准的标准合约条款的公司内部协议。当我们将您的资料传输到 Block 以外的组织时，我们使用欧盟委员会批准的标准合约条款来保护传输，除非我们将个人资料传输到已实施有约束力的公司规则或使用欧盟-美国隐私盾的第三方，在这种情况下，我们可能会依赖其中一种机制来保护传输。如需更多信息，包括如何获得用于保护您的信息的文件的副本，请按照下文“联系我们”部分中的说明与我们联系。

澳大利亚居民

我们可能会向位于澳大利亚以外的接收方披露您的个人信息。例如，这可能包括以下机构：

  • Block, Inc. 集团公司的其他成员；
  • 我们签订合约的国外运营的服务提供商；以及
  • 我们与之合作提供商品和服务的国外运营的组织，这些组织可能位于美国。

如果我们采取上述做法，我们将确保落实适当的安排，以确保根据澳大利亚隐私法规保护您的个人信息。

中国居民须知

个人资料的收集和处理

如果您向我们提供关于其他个人（例如您的配偶、同居/同性伴侣和/或受抚养人）的个人资料（例如用于紧急联系、福利管理和其他必要目的），您有责任告知这些人员此类个人资料的相关权利，并且您可以出于上述目的向我们披露其个人资料。如果适用的中国法律要求获得此类同意，以收集、使用、存储、跨国传输及处理其个人资料，您还有责任获取这些个人的同意（除非您可以代表他们提供此类同意）。

根据适用的中国法律，您的银行详细信息、健康和医疗状况以及指纹可能会被确定为敏感个人资料。我们将处理上述敏感个人资料，仅用于本隐私声明“我们为何收集、使用和存储此信息”部分中指明的目的。

个人资料的共享

我们可能会与其他实体共享您的个人资料，并且用于本隐私声明“我们如何共享您的个人资料”部分中所述的目的。

倘若因合并、分立、解散、宣告破产或其他原因，我们必须传输个人资料时，我们将告知您接收方的名称和联系方式。接收方在处理您的个人资料时应遵守本隐私声明。

个人资料的跨境传输

Block 是一家跨国公司，在多个国家和地区拥有实体。出于本隐私声明“我们为何收集、使用和存储此信息”部分中所述的目的，我们可能会与中国以外的以下实体共享您的个人资料：

  • Block Inc. 公司集团的其他成员（可在此处获得我们的全球实体列表）；
  • 我们签订合约的国外运营的服务提供商；以及
  • 我们与之合作提供商品和服务的国外运营的组织，这些组织可能位于美国。

我们将使用合法的跨境传输机制将您的个人资料传输到国外，并采取必要措施，确保国外接收方可以提供与适用的中国法律要求的相同级别的保护。

个人资料的保留

除非适用的中国法律另有要求，否则我们会保留您的个人资料，直至完成隐私声明中所述收集目的所必需的期限。

我们保持适当的技术和组织措施，使我们能够保护您的个人资料的安全性和保密性。如果发生安全事件，在适用的中国法律要求时，我们将立即采取补救措施并通知相关政府机关和受影响的员工。

IT 系统的监测和使用

我们为您提供信息技术 (IT) 系统、工具、电子设备和虚拟专用网络 (VPN)（简称“Block IT 系统”），用于开展公司相关业务。鉴于 Block IT 系统归 Block 所有，您通过此类 Block IT 系统访问、存储或传输的所有资料都可能经我们处理。Block 有权访问、审查、监测、成像、向国外传输、搜索或删除存储在 Block IT 系统上的任何资料或应用程序，或出于人力资源管理、业务运营、信息和其他安全运营、内部调查、审计和风险管理、遵守我们的法律义务或任何其他合法目的，随时访问、监测或搜索工作场所。

您的权利

对于我们维护的您的个人资料，您拥有适用的中国法律允许的相关权利，例如：

  • 访问您的个人资料。您有权访问您的个人资料，适用法律法规规定的除外；
  • 限制或反对。您可以要求我们限制使用您的个人资料，或者反对我们在某些情况下使用您的个人资料；
  • 更正您的个人资料。当您发现我们处理的您的个人资料有任何错误时，您有权要求我们作出更正；以及
  • 删除您的个人资料。如果我们对个人资料的处理违反法律法规，或者如果我们收集、使用或处理本隐私声明中未涵盖的个人资料并且未经您的同意（或不依赖适用的中国法律允许的其他合法依据），您可以要求我们删除您的个人资料。

如果您希望行使您的任何权利，请通过[电子邮件地址]与我们联系。在核实您的身份后，我们会及时与您联系以处理您的请求。

本隐私声明的更新

本隐私声明可能会定期更新。我们将相应地更新本隐私声明顶部的日期，并鼓励您查看本隐私声明的更改，登录 Blockup.com/legal 可以查看更改。在某些情况下，根据适用法律的要求，我们也可能会主动告知您具体的资料处理活动或对本隐私声明的重大更改。
